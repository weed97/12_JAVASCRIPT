02. 태그 이름을 이용한 요소 노드 취득

        Document.prototype/Element.prototype.getElementsByTagName 메서드는
        인수로 전달한 태그 이름을 갖는 모든 요소 노드들을 탐색하여 반환한다.
        메서드 이름의 Elements가 복수형인 것에 알 수 있듯 여러개의 요소 노드 객체를 갖는 DOM
        컬렉션 객체인 HTMLCollection 객체를 반환한다.
